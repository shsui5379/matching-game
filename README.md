# matching-game
A picture matching game HTML template.  Provide your own images

# instructions
create a folder named: images
inside the images folder, include 8 picutures.  name them this way: facex.jpg   where x is a number from 1-8.  ex: face1.png, all the way up to face8.png
then, include a card backside image.  name it: back side.png
